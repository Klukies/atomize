## Installation through a CDN

Cookie though is packaged to be used directly in the browser, which means it doesn't require any build or tools:

{/* TODO: add a Code component */}

```html
<script type="module">
  import { init } from 'https://unpkg.com/cookie-though';

  // Initialise the library with or without a configuration
  init();
</script>
```

Install Cookie Though from your terminal via npm or yarn.
